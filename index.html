Notiser
Intern handbok

Svar på vanliga frågor från team-medlemmar.

Hur svarar jag på en supportförfrågan?

Inom 24 timmar på vardagar. Använd mallen "Supportsvar — generell" i Mallar-fliken.

Var vänlig, tydlig och kortfattad. Ange alltid om problemet är löst eller om vi följer upp.

Hur lägger jag till en ny rabattkod?

Gå till Rabatter-fliken. Fyll i kod, %, och beskrivning. Klicka "Lägg till kod".

Koden synkas automatiskt till hemsidan. Be alltid om godkännande av teamledaren innan du ger > 30 % rabatt.

Vilken AI-modell använder vi?

Internt: Claude Sonnet via Anthropic-API för coachen i appen.

Backup: OpenAI GPT-4 om Anthropic är nere. Aldrig leverera annat än sport-relaterade svar (strikt filter).

Hur prioriterar jag klubbkontakter?

Använd Todo-fliken för att hålla koll. Prioritera klubbar som:

1. Stora friidrottsklubbar (Bauhausgalan-deltagare, m.fl.)
2. Klubbar som hört av sig via Formspree
3. Geografisk närhet till våra fysiska event

Var hittar jag användares profiler/data?

Supabase-dashboard → users-tabellen. Filtrera på email eller user_id.

Använd ALDRIG service-role nyckeln i klienten — bara anon key (public).

Vid GDPR-begäran om radering: använd "delete user" i Supabase Auth + radera relaterade rader manuellt.

När lanseras appen publikt?

Q3 2026. Waitlist är öppen. Endast inbjudningar tills lansering.

Vid frågor: hänvisa till nyhetsbrev-anmälan på hemsidan för 50 % rabatt första året.

Vad ska jag göra vid en betalningstvist (Stripe)?

1. Logga in på Stripe Dashboard → Disputes-fliken
2. Kolla orderhistorik i Supabase
3. Svara inom 7 dagar med leveransbekräftelse + kund-kommunikation

Vid osäkerhet: kontakta team-ledaren innan svar.

Hur skickar jag nyhetsbrev?

För nu: använd Formspree-listan för waitlist + manuell utskick via Gmail.

Framtid: integrera Mailchimp/Resend när vi når 500+ prenumeranter.

Frekvens: max 1 mejl/månad. Bryt aldrig löftet i avtalsvillkoren.

Vart skickar vi tekniska bug-rapporter?

Skapa en Todo med tag BUG: + beskrivning. Inkludera:

1. Vad användaren gjorde
2. Vad som hände vs förväntat
3. Browser + OS
4. Skärmdump om möjligt

Triage minst 1 gång/vecka.

Vad räknas som "off-topic" för AI-coachen?

Allt utanför friidrott + relaterad träning. T.ex.:

❌ Politik, religion, ekonomi, allmänna livsfrågor
❌ Andra sporter (fotboll, hockey, etc.)
❌ Personliga råd ej kopplade till idrott
✅ Friidrottsteknik, träning, kost, mental förberedelse, skadeskydd, återhämtning
